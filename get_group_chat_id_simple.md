# Простой способ получить Chat ID группы Telegram

## Способ 1: Через n8n (самый простой)

1. Зайдите в n8n: https://ai.rahima-consulting.ru
2. Создайте новый workflow (или используйте существующий)
3. Добавьте узел **Telegram** → **Send Message**
4. В настройках узла:
   - Выберите вашего бота (или создайте новое подключение)
   - В поле **Chat ID** временно введите: `@+AJojH2tNS8Y3ZWVi` или просто попробуйте отправить сообщение
   - Если это не сработает, перейдите к способу 2

## Способ 2: Через Telegram API напрямую

### Шаг 1: Найдите токен бота

1. Зайдите в n8n: https://ai.rahima-consulting.ru
2. Найдите workflow, который отправляет сообщения в Telegram
3. Откройте узел Telegram
4. Посмотрите настройки подключения - там должен быть токен бота (формат: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

### Шаг 2: Получите Chat ID

После того как у вас есть токен бота:

1. Убедитесь, что бот добавлен в группу https://t.me/+AJojH2tNS8Y3ZWVi
2. Отправьте любое сообщение в группу
3. Выполните команду (замените `<BOT_TOKEN>` на токен):

```bash
curl "https://api.telegram.org/bot<BOT_TOKEN>/getUpdates" | grep -o '"chat":{"id":-[0-9]*' | grep -o -- '-[0-9]*' | head -1
```

Или откройте в браузере:
```
https://api.telegram.org/bot<BOT_TOKEN>/getUpdates
```

И найдите в ответе: `"chat":{"id":-1001234567890}`

## Способ 3: Использовать ссылку группы напрямую

В некоторых случаях можно использовать ссылку группы напрямую в n8n:
- Попробуйте в поле Chat ID ввести: `@+AJojH2tNS8Y3ZWVi`
- Или: `+AJojH2tNS8Y3ZWVi`

Но обычно нужен именно числовой Chat ID.
